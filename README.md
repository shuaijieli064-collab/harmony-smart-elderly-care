# Harmony Smart Elderly Care

基于鸿蒙分布式架构的社区智慧医养监护系统

## 项目概述

本项目针对当前智慧养老领域存在的系统数据割裂、强依赖硬件设备、适老化设计不足等问题，基于鸿蒙分布式架构提出轻量化解决方案，实现跨设备健康数据协同与智能监护。

## 系统架构

```
┌─────────────────┐
│   社区管理端(Web) │
└────────┬────────┘
         │
    ┌────┴────┐
    │         │
┌──┴──┐  ┌──┴──┐
│家庭端│  │个人端│
│ APP │  │ APP │
└─────┘  └─────┘
```

三级联动架构：社区-家庭-个人

## 核心功能模块

- 健康数据采集
- 智能提醒
- 异常预警
- 社区台账管理
- 亲情远程关怀

## 鸿蒙特色技术

- 服务卡片：桌面级健康概览
- 多设备流转：数据无缝接续
- 原子化服务：即用即走体验
- 分布式数据管理：跨设备数据同步

## 技术栈

- ArkUI
- ArkTS
- SQLite
- 分布式数据服务

## 研究目标

设计并实现一套基于鸿蒙分布式能力的社区智慧医养监护系统，实现健康数据在社区、家庭、个人三级主体间的安全共享与异常预警，并在真实社区环境中完成试点验证。

## 创新特色

1. **模式创新**：无硬件依赖的社区医养监护模式
2. **架构创新**：基于HarmonyOS的三级联动架构
3. **技术创新**：分布式数据管理 + 异常检测算法
4. **体验创新**：面向老年用户的极简适老化交互模型
